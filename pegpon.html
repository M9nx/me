PEGPON is an advanced Bash-based automation tool designed for comprehensive subdomain enumeration, fuzzing, and web reconnaissance. It leverages top-tier security tools like subfinder, assetfinder, ffuf, httpx, gau, waybackurls, dirsearch, katana, and many more to provide a complete bug bounty reconnaissance solution.

This powerful toolkit automates the entire reconnaissance process, from initial subdomain discovery to deep directory fuzzing and vulnerability identification. PEGPON streamlines complex workflows that would typically require manual coordination of multiple tools, making it an essential asset for security researchers and bug bounty hunters.

Created by @M9nx specifically for bug bounty hunting and penetration testing workflows, with continuous updates and improvements based on real-world testing scenarios.
